# אתר שירותים מקצועיים

אתר מודרני המציג שלושה שירותים עיקריים:
- אדריכלות נוף (Nature Design)
- מוצרי קוסמטיקה
- עיצוב גרפי

## תכונות

- ✅ תמיכה בעברית ואנגלית עם החלפת שפה
- ✅ עיצוב מודרני ויפה בהשראת NatureDesign
- ✅ תמיכה במובייל (Responsive)
- ✅ מעבר חלק בין טאבים
- ✅ גלריות תמונות עם אפקט hover
- ✅ סקשנים: Our Vision, How It Works
- ✅ פרויקטים: Kfir Reserve, Tropical Moses

## התקנה והרצה

פשוט פתח את הקובץ `index.html` בדפדפן.

או אם יש לך שרת מקומי:
```bash
# עם Python
python -m http.server 8000

# עם Node.js
npx http-server
```

ואז פתח בדפדפן: `http://localhost:8000`

## מבנה הקבצים

- `index.html` - המבנה הראשי של האתר
- `styles.css` - עיצוב האתר
- `script.js` - פונקציונליות הטאבים והחלפת שפה
- `backround_architect.avif` - תמונת רקע לכותרת אדריכלות נוף
- `Kfir Reserve/` - תיקיית תמונות לפרויקט Kfir Reserve
- `TROPICAL MOSES/` - תיקיית תמונות לפרויקט Tropical Moses

## החלפת שפה

האתר תומך בעברית ואנגלית. לחץ על הכפתור "עברית" / "English" בפינה העליונה כדי להחליף שפה. ההעדפה נשמרת ב-localStorage.

## הוספת תמונות

התמונות כבר מוצגות באתר תחת הפרויקטים:
- Kfir Reserve - 11 תמונות
- Tropical Moses - 4 תמונות

## עיצוב

האתר מעוצב בהשראת [NatureDesign](https://www.naturedesign.biz/) עם:
- צבעים טבעיים (ירוק, חום)
- רקע לבן נקי
- טיפוגרפיה מודרנית
- אפקטי hover על תמונות
